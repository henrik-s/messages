A small message service represented by a REST-API built with Flask

http://127.0.0.1:5000/
Schemes: http
Version: 0.1

Fetch all unread messages for an user

GET /user/{username}/messages/unread

Get all unread messages for a specific user. A message is considered to be read once it has been fetched once using this method.

in
path
200 OK

Fetch messages within an interval for an user

GET /user/{username}/messages?{start}{stop}

Fetch messages within an interval, defined by start and stop index, for a specific user. Messages are sorted in descending order based on timestamp.

Delete message for an user

DELETE /user/{username}/messages?{id}

Delete messages for an user identified by the 'id' parameter. To remove multiple message in one request, simply add multiple 'id' parameters. E.g. /user/foo/messages?id=1&id=4

404 Not Found
